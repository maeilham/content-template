# 매일함 Content Template

매일함(maeilham)에서 사용하는 콘텐츠 저장소 템플릿입니다.
이 저장소를 **Use this template** 으로 복제해서 자신만의 콘텐츠 repo를 시작할 수 있습니다.

## 어떤 콘텐츠를 담나요?

매일 1통의 메일에 들어갈 짧은 학습 콘텐츠를 마크다운으로 관리합니다.
면접 질문, 인사이트, 뉴스레터, 도메인 지식 등 어떤 종류여도 좋습니다.

## 디렉토리 구조

```
.
├── README.md              # 이 repo의 소개
├── CONTRIBUTING.md        # 콘텐츠 추가/검수 가이드
└── content/               # 콘텐츠 파일
    └── 0001-example.md
```

- 모든 콘텐츠는 `content/` 안에 둡니다.
- 파일명은 `NNNN-slug.md` 형식 — 4자리 숫자가 발송 순서가 됩니다.
- 새 콘텐츠를 추가할 때는 가장 큰 번호 다음 번호를 씁니다.

## 빠른 시작

1. 이 저장소를 **Use this template** 버튼으로 복제
2. `content/0001-example.md` 를 참고해 첫 콘텐츠 작성
3. 매일함에 repo 등록 요청

자세한 작성 규칙은 [CONTRIBUTING.md](./CONTRIBUTING.md) 참고.
